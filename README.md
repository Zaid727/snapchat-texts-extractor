**I'll try to make this as easy to understand as possible.**

Firstly, take the screenshots you wanna extract the texts from and put them in a single folder, name it whatever you wish to find it easily.
Then download both the files.
Run "snapss.py" and select the folder in which the images are in.
Then a text file named "chat_transcript.txt" will show in the folder of your images.
Then copy and paste the "cleaner.py" file in the folder of your images.
Then run "cleaner.py" and then another text file "cleaned_chat.txt" will appear.

There you can see the texts from the images in a clean formatted way.
